Overview of the Project
Project: Tic-Tac-Toe AI Game Using C++

Objective
The main objective of this project is to build an unbeatable single-player Tic-Tac-Toe game where a human player competes against an intelligent AI agent. The project focuses on implementing foundational Artificial Intelligence search algorithms, decision-making logic, and game state evaluations in C++.

Key Activities
Design and maintain a standard 3x3 interactive game board using dynamic arrays or vectors.

Implement standard game logic including turn switching, validation of empty spaces, and check-win configurations.

Develop an intelligent AI player using the Minimax Search Algorithm to evaluate future possible moves.

Optimize the recursive decision tree using Alpha-Beta Pruning to eliminate unpromising branches, significantly improving execution efficiency.

Incorporate depth-based heuristic scoring to ensure the AI prioritizes winning faster or extending a defensive play.

Create a clean, continuous command-line loop allowing replayability and turn selection options.
